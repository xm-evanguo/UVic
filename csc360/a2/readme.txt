ACS is a program in C which simulate the Airline Check-in System.

How to use ACS
Use "make clean" to delete the files "acs" "*.o", and "*.exe" under current directory.
Then use "make" to compile acs.c.
After compiling, entre "./ACS [input file name]" with input file to start the program.

Input file
An input file "test.txt" is included under directory p2.
